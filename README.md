## 项目结构


```
Flask_movie_project/    <-- 根目录
├── app                <-- Web目录
│   ├── admin           <-- 后台业务逻辑
│   ├── home            <-- 前端业务逻辑
│   ├── __init__.py
│   ├── models.py       <-- 数据模型文件
│   ├── static          <-- 存放静态文件
│   └── templates       <-- 存放模板文件
├── app.py
├── logs               <-- 存放日志文件
├── manage.py          <-- Flask应用的启动程序
├── movie.sql          <-- 数据库建表语句
└── req.txt            <-- 项目依赖包
```
